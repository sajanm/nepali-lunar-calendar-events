# Nepali Lunar Calendar Events
List of all the Lunar Calendar Events sorted by Date in JSON

Event Format:
```sh
{
    "holiday": boolean,
    "event": string,
    "tithi": string,
    "ad": string,
    "bs": string
}
```

Example:
```sh
{
    "holiday": true,
    "event": "नयाँ वर्ष ",
    "tithi": "तृतिया",
    "ad": "04/14/2017",
    "bs": "2074-01-01"
}
```
